Project description

infoportal. is a project aimed to a specific profile with a specific need. It started from an open and personal brief, in which I found a very personal problem.

I developed the complete UX and UI process of a digital app that helps people that want to live and work in Spain understand the process and adapt to bureaucracy, which is very difficult to do.

As the final project of the UX UI Ironhack Bootcamp, infoportal. is an app that categorizes, organizes and filtrates information so users can find exactly what they need to do, reducing the stress, anxiety and time consuming process to an easy and amicable one.
