Pace

Project description

Pace is an UX UI Design Case Study based on a Wellness Brief.

Through a complete UX and UI process, Pace aims to focus on the bad sleep problem, that people with very mentally intense lives unfortunately have.

As the first individual full project of the UX UI Bootcamp in Ironhack Madrid, the outcome of this project was a digital app that can help an user with this specific needs to find a time for themselves, to create a sleep ritual, to prepare and to be able to rest in a way that the mind and body can recharge for the very next day.
